## REQUIREMENTS
* Node JS
* Mongo DB

## DEPENDENCIES
* express
* mongoose
* body-parser

## HOW TO RUN LOCALLY
* Clone files from GitHub. <br>
* Install all dependencies. 
> npm install
* Run MongoDB server
> mongo
* Run from command prompt
> node index.js
* Navigate to `http://localhost:3000`.
